Standalone Examples » Tooltips

This show how to have a tooltip by specifying a custom string per data item. 

var pie = new d3pie("pie", {
  header: {
    title: {
      text: "A Very Simple Pie",
      fontSize: 30
    }
  },
  data: {
    content: [
      { label: "JavaScript", value: 264131, caption: "JS Rocks! Woo!" },
      { label: "Ruby", value: 218812, caption: "Ruby is foreign and weird" },
      { label: "Java", value: 157618, caption: "Old and verbose (buuuuurn!)"}
    ]
  },
  tooltips: {
    enabled: true,
    type: "caption"
  }
});
