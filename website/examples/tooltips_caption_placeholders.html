Standalone Examples » Tooltips #X: caption placeholders

To provide a little more "live" data within your tooltips, you can enter any of the following placeholders: {label}, {value}, {percentage}. They will be replaced by the 

var pie = new d3pie("pie", {
  header: {
    title: {
      text: "Tooltips: example #1",
      fontSize: 30
    }
  },
  labels: {
    inner: {
      format: "none"
    }
  },
  data: {
    content: [
      { label: "JavaScript", value: 1, caption: "Javascript: taking over the universe." },
      { label: "Ruby", value: 2, caption: "A language I should know more about." },
      { label: "Java", value: 3, caption: "Old and verbose (buuuuurn!)" }
    ]
  },
  tooltips: {
    enabled: true,
    type: "caption"
  }
});
