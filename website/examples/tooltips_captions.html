Standalone Examples » Tooltips #1

This shows how to display tools tips on your pie chart segments, but supplying an additional caption attribute in your data. If you don't want have additional text for your tooltips, check out Tooltips #2. 

var pie = new d3pie("pie", {
  header: {
    title: {
      text: "Tooltips: example #1",
      fontSize: 30
    }
  },
  labels: {
    inner: {
      format: "none"
    }
  },
  data: {
    content: [
      { label: "JavaScript", value: 1, caption: "JS Rocks! Woo!" },
      { label: "Ruby", value: 2, caption: "Ruby is foreign and weird" },
      { label: "Java", value: 3, caption: "Old and verbose (buuuuurn!)"}
    ]
  },
  tooltips: {
    enabled: true,
    type: "caption"
  }
});
